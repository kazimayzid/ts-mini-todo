# Mini Todo Project by TypeScript

A simple, browser-based Todo application built with **TypeScript**, **HTML**, and **vanilla JavaScript**.  
This project demonstrates CRUD operations (Create, Read, Update, Delete) and persistent storage using **localStorage**.

---

## Features

- Add new todos
- Mark todos as completed with a checkbox
- Edit todos inline
- Delete todos
- Persist todos in localStorage
- Fully written in TypeScript for type safety
